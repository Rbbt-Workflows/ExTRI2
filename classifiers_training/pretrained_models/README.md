# Commands to install the used pretrained models:
```
git lfs install

# distilbert (used for debugging)
git clone https://huggingface.co/distilbert/distilbert-base-uncased

# BioLinkBERT
git clone https://huggingface.co/michiyasunaga/BioLinkBERT-base

# PubMedBERT
git clone https://huggingface.co/microsoft/BiomedNLP-BiomedBERT-base-uncased-abstract-fulltext 

# BioBERT
git clone https://huggingface.co/dmis-lab/biobert-base-cased-v1.2 

# SciBERT
git clone https://huggingface.co/allenai/scibert_scivocab_uncased 

# PubMedBERT-large
git clone https://huggingface.co/microsoft/BiomedNLP-BiomedBERT-large-uncased-abstract
```
